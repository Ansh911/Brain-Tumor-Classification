# Brain Tumor Classification using CBAM-Integrated ResNet50

## Overview
This project classifies brain tumor MRI images using deep learning.

## Models
- ResNet50 (transfer learning)
- MobileNet
- ResNet50 + CBAM (Convolutional Block Attention Module)

## Features
- MRI image classification
- Transfer learning with pretrained models
- CBAM attention mechanism
- Accuracy evaluation and confusion matrix generation
- Training history visualization

## Tech Stack
- Python
- PyTorch
- torchvision
- NumPy
- Matplotlib
- scikit-learn

## Dataset Structure
Training/
Testing/

## Run
1. Install dependencies
2. Update dataset path
3. Run:

python brain_tumor_cbam_resnet50.py

## Resume Description
Developed a Brain Tumor Classification system using a CBAM-enhanced ResNet50 architecture on MRI datasets. Implemented transfer learning, attention mechanisms, model evaluation, and performance comparison against baseline CNN architectures.
